# Simple Purchase Example

The "Hello World" of AgentGate — a complete agent transaction in under 20 lines.

## Run it

```bash
pnpm install
pnpm start
```

## What it does

1. Creates an AgentGate client in sandbox mode
2. Registers an agent with spending policies
3. Executes a purchase transaction
4. Prints the result

No API keys needed — sandbox mode uses the built-in mock adapter.
